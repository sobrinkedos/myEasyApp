# @restaurant-system/mobile

Componentes React Native para mobile do Restaurant Design System.

## Instalação

```bash
npm install @restaurant-system/mobile @restaurant-system/tokens react react-native styled-components
```

## Uso

```tsx
import { Button } from '@restaurant-system/mobile';

function App() {
  return (
    <Button variant="primary" size="medium">
      Clique aqui
    </Button>
  );
}
```

## Componentes Disponíveis

Os componentes serão adicionados conforme o desenvolvimento progride.

## Compatibilidade

- React Native >= 0.72.0
- React >= 18.2.0
