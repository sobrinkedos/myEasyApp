## Uso

```typescript
import { colors } from '@restaurant-system/tokens';

// Usar cor primária
const primaryColor = colors.primary[500]; // #FF7A4D

// Usar cor de feedback
const successColor = colors.feedback.success; // #22C55E

// Usar cor de status
const pendingColor = colors.orderStatus.pending; // #F59E0B
```
