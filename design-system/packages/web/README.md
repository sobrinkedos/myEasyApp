# @restaurant-system/web

Componentes React para web do Restaurant Design System.

## Instalação

```bash
npm install @restaurant-system/web @restaurant-system/tokens react react-dom styled-components
```

## Uso

```tsx
import { Button } from '@restaurant-system/web';

function App() {
  return (
    <Button variant="primary" size="medium">
      Clique aqui
    </Button>
  );
}
```

## Documentação

Acesse o Storybook para ver todos os componentes disponíveis, suas props e exemplos de uso.

```bash
npm run storybook
```
