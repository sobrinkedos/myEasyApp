# Product Overview

Restaurant API Core - Backend system for integrated restaurant, bar, and snack bar management.

## Core Features

- Multi-tenant restaurant management system
- User authentication with JWT and 2FA support
- Role-based access control (RBAC) with hierarchical permissions
- Cash register and session management
- Product catalog with categories and ingredients
- Inventory/stock management with transaction tracking
- Table and command management
- Order processing and tracking
- Audit logging for compliance

## Target Users

- Restaurant owners and managers
- Waiters and kitchen staff
- Cashiers and treasurers
- System administrators

## Infrastructure

- Cloud-hosted PostgreSQL (Neon)
- Cloud-hosted Redis (Upstash)
- RESTful API with Swagger documentation
- Docker containerization support
